# CensGSTCAR
Function for running MCMC model for left censored areal count data with spatiotemporal dependencies

See <i>Example.R</i> for how to use the function for running an MCMC. It also includes a synthetic data generation mechanism which is appropriate for using with this function.

See <i>MainFunc.R</i> for the details of the MCMC function
